A plugin for displaying the astrological position of planets both in tropical and sidereal zodiac, Indian Panchang and the phase of Moon.

== Description ==

<h3>Planetary Position</h3>This plugin displays the current UTC / GMT panchang details, planetary positions both tropical and sidereal, and phase of moon. Additionally, the positions of planets is displayed in 'wheel' of zodiac signs / nakshatras or lunar mansions.

= Using in theme =
The plugin settings are 
a) Show the Panchang
b) Show the Planetary Positions table of sidereal and tropical zodiac
c) Show the 'wheels' of tropical and sidereal zodiac
d) Show the current phase of moon.
e) Select Ayanamsa (default is True Chitrapaksha)

== Installation ==

1. Navigate to the 'Add New' Plugin Dashboard
2. Select 'planetary position' from wordpress "Search Plugins"
3. Install the 'planetary position' plugin
4. Activate the plugin on the WordPress Plugin Dashboard
5. Place the 'Planetary Position Widget' into a Widget area through the 'Appearance -> Widgets' menu in WordPress. Alternately one can place it in a page or post by using "[planets]" shortcode. Place the Planetary Position widget wherever in sidebar your wish to put.
